# web-eventos-publicos
Una entidad local va a organizar un evento abierto al público y necesita una web informativa para difundirlo.

**Este proyecto trata sobre la creación de una web informativa y difusora de un evento público ficticio.**

La web debe:
  - Presentar el evento de forma clara y atractiva.
  - Mostrar la información esencial de todo.
  - Todo el contenido debe verse bien; debe ser una página responsive.

El evento es libre, pero debe ser realista. Debe tener un aspectocultural.

El diseño de la web estática se realizó usando Bootstrap y ha sido construido y gestionado mediante GitHub.

Por último, se ha instalado y configurado un cliente de correo electróncio mediante el cual se enviará la URL
del repositorio.
